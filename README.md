## TRPG 回放

点击主界面右上角的“🎬 回放”打开回放工作室。DM 可以选择公开、指定队伍、指定玩家或 GM 范围，开始实时录制，也可以从当前战役的既有聊天生成回放。选择 24、30 或 60 FPS 后点击“渲染并导出 MP4”，应用会隐藏编辑器界面、逐帧渲染场景与台词层，并通过 PATH 中的 FFmpeg 输出 H.264 MP4。DM 的角色立绘和姓名显示在左侧，玩家显示在右侧；立绘保持原始比例并贴底展示。

公开回放只包含公开消息；队伍和玩家回放在写入事件前执行访问检查。GM 回放可能含私聊和隐藏内容，界面会显示发布警告。停止播放或完成视频渲染后会恢复 DM 播放前的体素场景。可选的 `*.willow-replay.json` 只用于保存和再次编辑回放项目，不是最终交付格式。
